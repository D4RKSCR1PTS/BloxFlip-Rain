How to get BloxFlip Token: 
go to inspect element (f12)
go to console
paste this: localStorage.getItem('_DO_NOT_SHARE_BLOXFLIP_TOKEN')
a long string of content should show up, right click it and then click "Copy string contents"
you now have your BloxFlip Token, paste it in config and the script should work properly.
